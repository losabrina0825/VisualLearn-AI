# VisualLearn AI: Transforming Physics into Visual Logic

**VisualLearn AI** is an interactive educational tool designed to bridge the gap between abstract physics formulas and conceptual understanding. Developed for the **Build with Medo Hackathon**, this project focuses on deconstructing complex equations like $F = mg$ into intuitive, visual, and human-language explanations.

## 💡 The Problem
Traditional physics education often forces students to memorize static formulas (e.g., $F = mg - R$). For many beginners—including myself as a first-year engineering student—the "visual logic" behind these symbols is missing. Why is it a subtraction? Why do we multiply mass and gravity? 

**VisualLearn AI** solves this by acting as a "Visual Translator."

---

## ✨ Key Features

### 1. Theory Breakdown (The "Why")
Instead of just showing letters, users can click on each component of the formula ($F$, $m$, $g$) to receive a "Plain English" explanation of its physical meaning.

### 2. Step-by-Step Pedagogical Flow
* **Step 1: Theory** - Master the formula components.
* **Step 2: Simple Case** - Observe free fall in a vacuum (Gravity only).
* **Step 3: Comparison** - Run side-by-side experiments comparing Vacuum vs. Atmosphere to see how Air Resistance physically impacts motion.

### 3. Real-Time Data Analysis
A built-in data panel provides a logical comparison between scenarios, showing how forces interact (e.g., $Total Force = Gravity - Resistance$) to determine the final speed.
